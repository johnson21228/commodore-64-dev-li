# Kick Assembler Smoke Example

This is a source-only starter for a tiny Kick Assembler experiment.

The exact build command depends on where `KickAss.jar` is installed. Capture the local command before treating it as part of the Workbench.

Common command shape:

```bash
java -jar /path/to/KickAss.jar smoke.asm -o build/smoke.prg
```

Then run the `.prg` in an emulator and capture evidence.
