# Memory Pal App

Lab 008 is the first product-shaped app in the C64 Learning Lab.

It is a tiny rule-based helper that answers a few C64 memory questions. It intentionally stays native to the machine: menu input, short text answers, and no server dependency.

This is the safe local predecessor to later experiments such as:

```text
C64 Memory Pal with saved notes
C64 Memory Pal as a terminal to a host gateway
C64 LLM terminal through a modern server
```
