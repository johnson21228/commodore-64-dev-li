# First C64 Learning Lab Curriculum Batch

This capture extends the lab sequence from three low-level memory examples into a small curriculum.

Included labs:

```text
Lab 004 PETSCII UI
Lab 005 Keyboard Input
Lab 006 Sprite Basics
Lab 007 SID Tone
Lab 008 Memory Pal
```

The sequence now covers:

```text
screen output
screen memory
color memory
text UI
keyboard input
sprites
SID sound
small product-shaped app
```

The batch preserves the Learning Lab rule: each lab should be small, runnable, emulator-first, and easy to explain.
