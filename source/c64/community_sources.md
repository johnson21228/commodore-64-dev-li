# C64 Community and Corpus Sources

This source map names places to investigate for C64 Learning Lab ideas and patterns. It does not replace current web research when making claims about tool availability, hardware availability, or project status.

## Candidate sources

```text
Lemon64 forums
Lemon64 game database and comments
C64 Codebase
r/c64
VICE documentation
cc65 documentation
Kick Assembler documentation
C64 Studio / CBM prg Studio materials
C64 demo-scene writeups
BBS and WiC64 community pages
```

## How to use these sources

Use them to identify:

```text
beginner project archetypes
common toolchain setup paths
interesting C64 machine features
community vocabulary
known pitfalls
examples worth recreating as labs
```

## Corpus output shape

A research pass should classify ideas by:

```text
beginner / intermediate / advanced
toolchain required
machine feature taught
emulator-only vs hardware-needed
Workbench fit
possible first visible artifact
```

## Boundary

Do not paste large forum content or proprietary source into this repo. Capture summaries, citations, and links only when allowed and useful.
