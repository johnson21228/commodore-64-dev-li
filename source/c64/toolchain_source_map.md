# C64 Toolchain Source Map

This file captures the first source map for Commodore 64 development tooling. Check current official docs before installing or changing tools.

## cc65

- Project docs: `https://cc65.github.io/doc/`
- C64-specific docs: `https://cc65.github.io/doc/c64.html`
- Role in this Workbench: C compiler, assembler/linker toolchain, and a practical first `.prg` path.

## VICE

- Project site: `https://vice-emu.sourceforge.io/`
- Manual table of contents: `https://vice-emu.sourceforge.io/vice_toc.html`
- Role in this Workbench: emulator-first validation, command-line autostart, screenshots, and monitor/debugging workflows.

## Kick Assembler

- Project site: `https://theweb.dk/KickAssembler/`
- Manual PDF: `https://theweb.dk/KickAssembler/KickAssembler.pdf`
- Role in this Workbench: assembly-first experiments and demos.

## C64 Studio

- Project repository: `https://github.com/GeorgRottensteiner/C64Studio/`
- Role in this Workbench: Windows-oriented IDE path for assembly/BASIC/game-development workflows.

## Capture guidance

When a tool is installed or used, capture:

```text
tool name
version command and output
install method
host OS
first successful build command
first emulator run command
artifact path
observed result
```
