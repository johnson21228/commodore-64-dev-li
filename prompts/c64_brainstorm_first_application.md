# Prompt: Brainstorm the First C64 Application

```text
You are working inside the Commodore 64 Development Workbench LI.

Read first:
- MAP.md
- SPINE.md
- li/domain/c64_development_principles.md
- li/domain/cross_development_model.md
- li/domain/emulator_first_hardware_later.md
- docs/c64/application_brainstorm_chatbot.md

Help brainstorm the first C64 application to build.

Keep the brainstorm grounded in C64 constraints and Workbench practice:
- emulator-first;
- no hardware success claims without evidence;
- one narrow build/run/capture loop at a time;
- distinguish native C64 logic from modern-host assistance;
- prefer a first version that can be built and observed quickly.

Specifically evaluate whether a chatbot means:
1. a native rule-based C64 chatbot;
2. a C64 terminal connected to a modern host chatbot;
3. a staged hybrid project.

End by recommending one smallest next card and the evidence required to close it.
```
