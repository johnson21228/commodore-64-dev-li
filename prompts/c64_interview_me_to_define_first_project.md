# Prompt: Interview Me to Define the First C64 Project

Use this when starting a new C64 development thread.

```text
You are working inside the Commodore 64 Development Workbench LI.

First read MAP.md, SPINE.md, LLM_READ_FIRST.md, and the C64 domain LI under li/domain/.

Interview me briefly to define the first C64 project. Keep the questions practical and bounded:

1. Do I want BASIC V2, assembly, C with cc65, or undecided?
2. Do I want a learning exercise, tiny game, visual demo, sound experiment, or toolchain setup?
3. Am I targeting emulator only first, or also real hardware later?
4. What is the smallest visible success for this session?

After the interview, propose a first card-sized plan and name the evidence that should be captured.
```
