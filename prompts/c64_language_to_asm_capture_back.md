# Prompt: Capture Back C64 Language-to-Assembly Evidence

Use this prompt after running the language-to-assembly lab.

Capture:

```text
1. The exact program.lang content.
2. The generator command that produced generated_intent.json and generated.s.
3. The build command and result.
4. The emulator or hardware observation.
5. Any mismatch between intended command behavior and screen evidence.
```

Do not mark the lab complete unless the generated PRG was built and reviewed.
