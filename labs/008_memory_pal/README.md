# Lab 008: Memory Pal

Purpose: turn the first concepts into a tiny interactive app.

Memory Pal is not a modern LLM chatbot. It is a small rule-based C64 helper that answers a few C64 memory questions from a menu.

Build:

```bash
cd /Users/stevejohnson/Developer/commodore-64-dev-li
make lab008
```

Run:

```bash
cd /Users/stevejohnson/Developer/commodore-64-dev-li
make lab008-run
```

Learning point: this is the first tiny product-shaped C64 app in the lab sequence. It combines text UI, keyboard input, and memory facts.
