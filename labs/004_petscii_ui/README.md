# Lab 004: PETSCII UI

Purpose: make the screen feel like a C64 interface before introducing custom graphics.

This lab uses `conio.h`, cursor positioning, color, and reverse video to draw a simple PETSCII-style panel.

Build:

```bash
cd /Users/stevejohnson/Developer/commodore-64-dev-li
make lab004
```

Run:

```bash
cd /Users/stevejohnson/Developer/commodore-64-dev-li
make lab004-run
```

Learning point: early C64 apps can feel rich with text, placement, color, and reverse video before sprites or bitmaps are needed.
